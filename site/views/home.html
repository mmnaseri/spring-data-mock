Spring Data Mock

Spring Data Mock is a framework for mocking Spring Data repositories.

Download

You can download this framework via:

  • Github

  • Maven Central by adding this dependency to your pom.xml file:

        
        <dependency>
            <groupid>com.mmnaseri.utils</groupid>
            <artifactid>spring-data-mock</artifactid>
            <version>${latest-version}</version>
            <scope>test</scope>
        </dependency>

  • By cloning the project and installing it locally:

    git clone https://github.com/mmnaseri/spring-data-mock.git
cd spring-data-mock
mvn install

    and then adding the same dependency as above, or just grabbing the JAR file locally from the target folder.

Future

The future of this project depends highly on the availability of contributors.

At the moment I am maintaining this myself since this is something I actively use (that is actually how this framework was born to begin with).

Donating certainly doesn't hurt in my esteem of the project and how much time I dedicate to it.

However, there is a road map that I keep, just to stay focused when I do any development work on this project.
